# 電商前台-後端評估需求用

## 專案環境與版本

```
NodeJS v16.17.0
vue/cli 5.0.8
```

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npx tailwindcss -i ./src/assets/css/input.css -o ./src/assets/css/app.css --watch
npm run serve --mode dev
```

### Compiles and minifies for production

```
npm run build --mode prod
```
